  COMMON ISSUES & SOLUTIONS
═══════════════════════════════════════════════════════════════

Issue: 404 error on /api/organizer/checkpoint-teams
Solution: Hard refresh browser (Ctrl+Shift+R) - it's cached

Issue: 503 error on /api/vapid-public-key
Solution: VAPID keys not set in environment variables
         Go back to Variables tab and add them

Issue: "Unexpected token '<'"
Solution: Server returned HTML error page instead of JSON
         Check deployment logs for actual error

Issue: Service won't start
Solution: Check logs in Railway/Render dashboard
         Usually missing environment variables

Issue: Can't login
Solution: Check MongoDB connection string is correct
         Default password is "codehunt" for teams
